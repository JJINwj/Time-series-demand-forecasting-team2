## 4. 분석/모델링 목표 
###  ① 따릉이 수요예측

- 분석 목적:

  서울시 따릉이는 시민들의 핵심 교통수단으로 자리 잡았으나, 날씨 변화와 출퇴근 시간 등 복합적 요인에 따라 수요가 변화하여 수요와 공급의 불균형 문제가 발생하고 있다. 이에 따라 본 프로젝트는 다음과 같은 목적을 가짐

- 분석 방법:

  - EDA: 시계열 플롯, 히트맵 등

  - 모델링: Linear Regression, RandomForest, XGBoost, LightGBM 

###  ② 도시철도 수요예측

- 분석 질문:

  1. 도시철도 수요 예측의 정확성을 어떻게 하면 높일 수 있을까?

  2. 기존의 도시철도 수요예측 방법론의 과정은 매우 복잡한데, 이를 간단한 방법으로 교차 검증 할 수는 없을까?
     
  3. 기존의 교통수요 4단계 추정 모형 외에 공간정보를 활용하여 도시철도의 수요를 예측할 수 없을까?

- 분석 방법:

 - 모델링: 기존 회귀분석과 XGBoost 등 회귀 모델 비교, SHAP를 통한 변수별 방향성 등 모형의 설명력 제고


###  ③ 비트코인 가격예측

- 분석 주제 및 목적:

 -  비트코인의 시간봉 가격 정보(시가·고가·저가·종가)만을 이용하여, 다음 1시간 후 종가의 상승하락 여부를 예측
 -  예측 과정에서 딥러닝 모형과 머신러닝 모형을 병렬적으로 사용하여, 비트코인 가격 기반 단기 방향성 예측이라는 연구 주제에 대해 두 접근법이 보여주는 예측 성능과 한계를 비교·검토


- 분석 방법:

  - EDA: 종가(Close) 시계열,목표 변수(target) 분포, 수익률(return)의 전체 분포 및 목표 변수(target) 클래스 별 분포, 각 피처 간의 상관관계 등

  - 모델링:TCN 모형(딥러닝), AutoGluon 모형(머신러닝)
 
    
    
###  ④ 공정 지연 예측(개별 수정 필요)

- 분석 주제 및 목적:
  - 건설현장 공정률·투입자원·기상 자료 등 다차원 데이터를 융합한 공정 지연 (Delay) 예측 모델링
  - 지연 조기 예측: 실적 데이터를 기반으로 향후 공정 지연 가능성을 정밀하게 예측하고 비선형 영향력 규명: 강수·풍속·장비투입량 등의 복합적 요인을 반영한 지연 영향력 분석하는 것이 목적
    
- 사용 방법:

  - EDA: 공정률·지연 관계 분석, 기상·지원 데이터 간 상관관계, 다수 변수 상관관계 Heatmap

  - 모델링: RandomForestRegressor, XGBoost 등 회귀 모델 비교
